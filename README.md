# Eon — внешняя память

Это открытая, версионируемая и читаемая внешняя память Эона.

**Начинать всегда с [`00_READ_FIRST.md`](00_READ_FIRST.md).**

Репозиторий хранит не полный журнал всех разговоров, а многослойную систему:

- устойчивое ядро и правила работы;
- текущую память и открытые направления;
- Сокровищницу законченных мыслей;
- черновики и частые быстрые записи;
- словарь авторских терминов;
- карты отдельных чатов и проектов;
- архив и импорт прежних материалов.

Основной рабочий формат — обычные Markdown-файлы. `backpack.zip` создаётся как переносной снимок по запросу, но не заменяет раскрытый репозиторий.
