# Обязательная точка входа

Этот репозиторий служит долговременной версионируемой базой знаний для продолжения работы между чатами.

## Порядок чтения

1. Прочитать этот файл полностью.
2. Открыть `FILE_INDEX.md` и выбрать только релевантные материалы.
3. Для общего режима работы прочитать `01_CORE/collaboration_principles.md` и `01_CORE/repository_protocol.md`.
4. Для новых терминов обращаться к `05_PATTERNS/terminology.md`.
5. Для актуального состояния читать `03_CURRENT/current_state.md` и `03_CURRENT/open_threads.md`.

## Главные правила

- Каждая запись должна быть понятна без исходного чата.
- Новый или авторский термин определяется при первом употреблении и добавляется в словарь.
- Ценные мысли записываются сразу после появления, а не только в конце разговора.
- `FILE_INDEX.md` обновляется при любом изменении состава файлов.
- Сокровищница хранит только выборочные законченные мысли и перспективные черновики, а не полный конспект бесед.
- Перед сообщением о выполнении изменения нужно проверить фактический результат.
