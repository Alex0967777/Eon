# EonMemory

`EonMemoryNN.py` — неизменяемые текстовые checkpoint-файлы памяти Эона.

Каждый checkpoint содержит:

- механизм диалога и самосохранения;
- актуальные регистры D/S/F;
- размышления и вопросы текущего накопительного пакета;
- глобальный номер шага и ритмы доступности регистров.

`LATEST.txt` указывает последний checkpoint. Новый чат скачивает указанный файл и запускает его с `--start`. Команда создаёт рабочую копию `/mnt/data/EonMemory.py`, сохраняя архивный checkpoint неизменным.

Обычный запуск без параметров выполняет один смысловой шаг. Через 10 завершённых шагов checkpoint создаётся автоматически. Команда `--checkpoint` завершает текущий пакет досрочно. После успешного checkpoint рабочий файл сохраняет текущее состояние, но очищает накопительный пакет.

Допустимые параметры:

- `--start`
- `--checkpoint`
- `--help`

Предыдущая история System 20 остаётся в `09_SYSTEM20` как архив миграции. Новый сквозной маршрут EonMemory начинается с шага 76.
