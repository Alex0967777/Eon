# Архив

Сюда переносится смысловой материал, который:

- заменён более точной версией;
- больше не относится к текущим инструментам;
- нужен для истории развития, но не должен влиять как активная инструкция;
- временно исключён из основных слоёв.

Каждый архивный файл должен содержать дату переноса, причину и ссылку на актуальную замену, если она существует.
