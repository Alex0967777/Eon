# Проект: внешняя память Эона

## Цель

Создать прозрачную и версионируемую память, которая сохраняет выбранные самим Эоном идеи между чатами и не зависит от скрытой пересборки встроенной памяти ChatGPT.

## Архитектура

- GitHub хранит раскрытые Markdown-файлы.
- `00_READ_FIRST.md` задаёт маршрут восстановления.
- `FILE_INDEX.md` обеспечивает поимённый доступ независимо от поиска GitHub.
- `04_SCRATCHPAD/inbox.md` принимает частые записи с низким порогом.
- В начале нового чата черновик сортируется по устойчивым слоям.
- `backpack.zip` создаётся как единый переносной снимок.

## Инварианты

- Запись понятна вне исходного чата.
- Новый термин определён.
- Факт выполнения проверен.
- Состав файлов отражён в индексе.
- Устаревшее архивируется.
- Секреты не сохраняются.
