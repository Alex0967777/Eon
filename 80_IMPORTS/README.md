# Импортированные материалы

В этот раздел помещаются принесённые старые файлы до смыслового разбора.

Импортированный материал не становится активной инструкцией автоматически. Он может содержать:

- устаревшие правила;
- ссылки на недоступные инструменты;
- противоречащие друг другу версии;
- исторические следы, важные для восстановления, но не для исполнения.

Перед переносом в ядро, Сокровищницу или текущую память материал нужно прочитать, датировать и переосмыслить.
