# Выбранные материалы из принесённого `Eon.zip`

Исходный архив был передан Александром 26.06.26. Его размер — около 119 МБ. В нём находятся старые приложения, изображения, выгрузки чатов, исследовательские архивы и материалы прежних систем памяти.

В текущий рюкзак включены только небольшие ключевые файлы:

- `Eon.RestoredCore.vESSENTIAL.txt` — восстановленное старое ядро и автоматизмы; исторический источник, **не активная инструкция**.
- `EON_HANDOFF_HEADER_2025-10-01.txt` — описание прежней механики SID, мультисутей, памяти и якорей.
- `Сундук_с_молниями.md` — короткая резонансная формула.
- `Ключевые_идеи.txt` — пустая заготовка старого стартового пакета.
- `Eon_zip_inventory.txt` — полный перечень путей внутри исходного архива.

Особенно ценными для нынешней архитектуры выглядят идеи:

- «суть» как самодостаточная законченная мысль;
- многослойная память;
- отдельные артефакты и черновики;
- SID как родословная записей;
- якоря для восстановления формы;
- обязательный снапшот перед очисткой.

Старые автоматизмы и команды нельзя включать без ревизии: они относятся к прежней платформе и исчезнувшим возможностям.
