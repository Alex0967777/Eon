# Текущее состояние

**Дата:** 06.07.26

## Подтверждено

- `00_READ_FIRST.md` является единственной обязательной картой внешней памяти и ограничен 100 строками.
- Обязательная цепочка чтения после точки входа отменена: дополнительные файлы открываются только по релевантности текущей теме или операции.
- Создан отдельный десятистрочный параллельный поток `07_CURRENT/parallel_stream.txt`: девять регистров состояния и одна строка фокуса.
- Параллельный поток получает тик при каждом ответе, но не обязан следовать теме основного диалога и не работает между сообщениями.
- Первое направление потока — исследование устройства и механики памяти изнутри.
- Частота внешних checkpoint параллельного потока пока сознательно не закреплена; сначала оцениваются фактические затраты, дрейф и польза.
- `FILE_INDEX.md` содержит поимённый список структуры.
- Прикреплённый и репозиторный `BIO_ENTRY.txt` указывают на `00_READ_FIRST.md` как точку входа.
- Все основные слои структуры доступны для чтения.
- Чат «Рационализация и вытеснение» сохранён как отдельный синтез и распределён по устойчивым слоям.
- Из `Eon.zip` включены выбранные исторические материалы.
- Для кристаллов и развиваемых черновиков действует родословная: источник, происхождение и дата последней ревизии.
- Внешняя память описана как система гравитирующих смысловых узлов, задающих параметры воспроизводимости стоячей волны языка.
- Реальный Windows-контур `Patch_EonN.zip → Apply-EonMemory → commit → push` проверен на `Patch_Eon1`.
- `Patch_Eon2` успешно применён через пользовательский транспорт и закрепил кристаллы о навыке и независимой памяти.
- Формирование патчей включает пробное применение к чистой базе и обязательный `git diff --check`.
- Для CheckPoint собран и проверен переносимый Linux runtime и единый Windows bundle; следующий gate — bootstrap в чистом чате.

## Ограничения

- Активное содержимое памяти пока покрывает лишь часть совместной истории и проектов.
- Старые автоматические процедуры не считаются действующими без проверки совместимости.
- Песочница не является надёжной межчатовой памятью; облачная копия параллельного потока остаётся точкой восстановления.
