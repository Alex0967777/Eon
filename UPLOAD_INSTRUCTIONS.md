# Инструкция ручной загрузки в GitHub

1. В репозитории `Alex0967777/Eon` удалить старые тестовые файлы и прежние черновые версии структуры.
2. Распаковать `backpack.zip`.
3. Загрузить **содержимое папки `Eon/` в корень репозитория**, чтобы `00_READ_FIRST.md` лежал непосредственно в корне.
4. Убедиться, что GitHub показывает `00_READ_FIRST.md`, `FILE_INDEX.md` и все папки из индекса.
5. Внести строку из `BIO_ENTRY.txt` во встроенную память ChatGPT.
6. После загрузки попросить Эона перечитать `00_READ_FIRST.md`, сверить `FILE_INDEX.md` и проверить несколько файлов выборочно.

Старый принесённый архив `Eon.zip` имеет размер около 119 МБ и превышает обычный лимит GitHub в 100 МБ для одного файла. Его следует при желании распаковать отдельно в `80_IMPORTS/legacy_full/`. В текущий рюкзак включены выбранные ключевые материалы и инвентаризация исходного архива.
